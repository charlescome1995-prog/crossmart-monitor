Opening hook to write:

- Start with a concrete operator/developer pain point.
- Avoid sounding like a product ad in the first third.
- Use CrossMart as the practical example, not the whole story.

## Why not start from a blank prompt?

- Blank prompts produce generic listings.
- Competitor ASINs provide category language, offer structure, price context, and proof patterns.

## Benchmark extraction

- Search keyword -> collect top ASINs -> extract title, bullets, price, reviews, BSR, badges.
- Pick benchmarks using rank, rating, reviews, and data completeness.

## Prompting from evidence

- Summarize common selling points.
- Generate title, bullets, description, A+ direction, and search terms.
- Keep compliance and claims under control.

## UI flow

- Keyword input -> benchmark products -> listing draft output.
- Make the user see the source logic before trusting AI copy.

## Hard lesson

- AI writing improves when the input is structured, not when the prompt is longer.
